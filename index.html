Smart Hangman

Description:
This is a simple Hangman game with the original rules, using words generated randomly and accesed trough an API. The smart part about the game is that after your attempt of winning, there will be a genetic algorithm that will try to guess the word.
A genetic algorithm applies the laws of real genetics trying to create more and more efficient genes in order to solve the given problem. It transmits information from generation to generation, through a process, which in the medical world is called meiosis, sometimes suffering mutations to increase diversity.
